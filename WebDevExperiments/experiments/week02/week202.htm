Bootstrap Tables

Description:

In this Experiment , I have implemented the Bootstrap tables. This uses different table css classes to style the rows and columns
and apply actions like hover on the rows and columns.

Also there are some contexual classes like active, success, info, warning, danger to color table rows according to the context.
There is also the responsice table class which makes the table scroll horizontally.
