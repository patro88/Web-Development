Focus and blur

On focus the color of the input box changes to yellow. On blur the color changes back to white.

Code Snippet :

  $("input").focus(function () {
   $(this).css("background-color", "yellow");
  });
  $("input").blur(function () {
   $(this).css("background-color", "white");
  });

fadeToggle and fadeTo methods :

FadeToggle : This method makes the selected element fade to hidden with an animation delay provided as input. The input can be fast, slow, or any integer value

FadeTo : This method makes the selected element fade to a value provided in the inputs. It takes two inputs, the transition speed as above, the other is the oapcity to which the element is faded to.



   $("#div1").fadeToggle()
   $("#div1").fadeToggle("slow")
   $("#div1").fadeToggle(3000)


   $("#div1").fadeTo("slow", 0.15)
   $("#div1").fadeTo("slow", 0.4)
   $("#div1").fadeTo("slow", 0.7)
