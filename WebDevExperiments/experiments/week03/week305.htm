JQueryUI Autocomplete and Tooltip

AutoComplete

The autocomplete also uses the scrollable options. Type in the textbox to see the options.

Code Snippet :

   $( "#tags" ).autocomplete({
    source: availableTags
   });

Desciption: The autocomplete method displays the autocomplete options while typing. The values which are displayed are stored in a variable and specified in the source option.



Jquery UI Tooltip

the tooltip used uses the property track : true to track the mouse movement.

Hover the field to see the tooltip.

Code Snippet :

   $( document ).tooltip({
    track: true
   });

Desciption: The tooltip method displays the tooltip. The track option is true. This means that the tooltip travels with the mouse.

Code Snippet :

  state = true;
  $("#responsive").click(function () {
   if (state) {
    $("#effect").animate({
     backgroundColor: "#aa0000",
     color: "#fff",
     width: 500
    }, 1000);
   } else {
    $("#effect").animate({
     backgroundColor: "#fff",
     color: "#000",
     width: 240
    }, 1000);
   }
  state = !state;

Desciption: This animation is done using the animate function of JQuery. In the options we specify the delay in milliseconds and the css after the effect. The function animates the transition from the initial state to final state in the specified time.
