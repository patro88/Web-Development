Jquery and Ajax

Description:

In this Experiment , I tried to load external content from server by ajax calls. I have used ajax load() method. Also I used load method with the optional callback function. The callback function has following parameters:

  • responseTxt - contains the resulting content if the call succeeds
  • statusTxt - contains the status of the call
  • xhr - contains the XMLHttpRequest object
In the function , I tested if the method returns a success or not. If it returns a success , I return a success alert message, Otherwise I return a failure alert.

Code Snippet :

Desciption: The ajax load method is used to load the file on the server. A callback is used to recieve the response and verify if the content is recieved or not.


  $("button").click(function () {
   $("#ajaxdiv").load("week3_ajax.txt", function (responseTxt, statusTxt, xhr) {
    if (statusTxt == "success")
     alert("External content loaded successfully!");
    if (statusTxt == "error")
     alert("Error: " + xhr.status + ": " + xhr.statusText);
   });
  });
