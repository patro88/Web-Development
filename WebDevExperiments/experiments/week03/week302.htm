Modify DOM using Jquery

Description:

In this Experiment , I have tried to modify the DOM elements. I used append(), prepend(), after(), before(), emmpty(), remove() for adding or removing elements.
Also I Learnt about val(), attr(), text(), html() for changing contents in DOM elements. I haven't used these in the experiments though.

Code Snippet :

  $("#appendItem").click(function () {
   $("ol").append("<li class='extra' > Appended item</li>");
  });

Description: The code is used to append the li to the ordered list. By appending, we add the element to souce list at the end.

  $("#prependItem").click(function () {
   $("ol").prepend("<li class='extra'>Prepended item</li>");
  });

Description: The code is used to prepend the li to the ordered list. By prepending, we add the element to teh source list at the start.

  $("#afterItem").click(function () {
   $("ol").after("<p class='extra'>After OL</p>");
  });

Description: The code is used to append the li after the ordered list. By doing this, we add the element after the end of the source element.

  $("#beforeItem").click(function () {
   $("ol").before("<p class='extra'>Before OL</p>");
  });

Description: The code is used to append the li before the ordered list. By doing this, we add the element before the start of the source element.

  $("#removeItem").click(function () {
   $("li").remove(".extra");
  $("p").remove(".extra");
  });

Description: The code is used to remove the desired element completely from the DOM.

  $("#emptyItem").click(function () {
   $("ol").empty();
  });

Description: The code is used to empty all contents of the desired element. This includes all its sub elements.
