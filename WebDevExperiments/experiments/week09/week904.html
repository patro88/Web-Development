Project: About me route and Fixed Footer

Description:

There are two main functioalities in this page. One the routing mechanism implemented and the other is fixed heder and footer. The routing is implemented using the ng-route attribute of the angular Js. I have not developed the design of the about page. Just added some content for now to test the route functionality.

In the Fixed header and footer design , I learnt and implemented teh position attributes and their functionality. I will be using these further in my application.

Code Snippet

Javascript Code: 
var app = angular.module("ATApp", ["ngRoute"]);
app.config(['$routeProvider',
  function ($routeProvider) {
      $routeProvider.
        when('/home', {
            templateUrl: 'carousalHome.html'
        }).
        when('/about', {
            templateUrl: 'aboutMe.html'
        }).
        otherwise({
            redirectTo: '/home'
        });
  }]);
In the above image I have tried to show the fixed header and footer by reducing the screen height and doing a scroll down. Note that the position values of both header and footer are fixed. This makes them fixed at the bottom or top of the page irrespective of the scrolling.
