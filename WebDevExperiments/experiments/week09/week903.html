Project: Responsive Design

Description:

This experiment is basically the display of responsice design. The home page was inspireed from teh bootstrap model. The bootstrap files are designed in a way to keep the pages displayed in all forms of electronic devices including the laptops, the pads and the mobile devices. The bootstrap uses the @media attributes to achieve this.

In persuit of responsive design and making my project accessible on all devices I started to work on responsive design from the start of the project.
Click here to visit the current version of the home page.

The image below is a screenshot of teh home page taken by minimizing the application to teh minimum level possible. I also have checked this application on the mobile device and all functionalities are working well





There is no special code written in css or html files to acheive teh resonsiveness as bootstrap handles it for me. However I have shared some of teh css code of my appliation here which helped in creating the pplication

CSS Code:

code for maintaining the navbar wrapper class which comes into effect only in small screens 

@media (min-width: 768px) {
  /* Navbar positioning foo */
  .navbar-wrapper {
    margin-top: 20px;
  }
 .navbar-wrapper .container {
    padding-right: 15px;
    padding-left: 15px;
  }
  .navbar-wrapper .navbar {
    padding-right: 0;
    padding-left: 0;
  }
