Project: Sign up and Register Modal Boxes

Description:

In this experiment I tried to design the sign up and register modal boxes. These are designed using the bootstrap features to show the look and feel of the buttons. Also, there is an additional why? page to show the details according to standards of web development. The sign in page contains the username or alias and the password along with teh remember feature for the browser cache. The register page has Details about the user like the email, the alias name and password to be entered twice to confirm.

The submit buttons in teh modal boxes are not workig now. The functionalities will be developed as I proceed in the project.
Click here to visit the current version of the home page.
