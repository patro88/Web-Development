Gradient Background :

There are 2 types of gradient backgrounds in css.

  • Linear gradients which go up/down/left/right/diagonally
  • Radial gradients defined by their center

Code Snippet :

 #lin_gradient {
  height: 200px;
  width: 300px;
  background: -webkit-linear-gradient(red, blue);
 }
  #rad_gradient {
  height: 200px;
  width: 300px;
  background: -webkit-linear-gradient(180deg, red, blue);
 }
