HTML5 media Elements


HTML video tag:

Code Snippet:

 <video src="http://v2v.cc/~j/theora_testsuite/320x240.ogg" controls>
  Your browser does not support ...
 </video>

Description:

This example plays a sample video, with playback controls, from the Theora web site.

HTML audio tag:

Code Snippet:

 <audio src="http://v2v.cc/~j/theora_testsuite/320x240.ogg" controls autoplay loop>
  Your browser does not support...
 </audio>

Description:

This code example uses attributes of the <audio> element:
  • control : Displays the standard HTML5 controls for the audio on the web page.
  • autoplay : Makes the audio play automatically.
  • loop : Make the audio repeat (loop) automatically.

Plugins:

The plugins can be added in the html page using object tag or the embed tag. The embed tag has no end tag.

Object tag:

Code Snippet:

 <object width="400" height="50" data="../images/car.swf"></object>

Embed tag:

Code Snippet:

 <embed width="400" height="50" src="../images/car.swf">
