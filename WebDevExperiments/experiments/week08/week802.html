Embedding youtube video

Description:

In this experiment , I have tried to embed a video into a iframe in html. This is required for my user page as he will be adding widgets onto iframes. This utility is very useful as it can be used to attach a video as well as play the video in the iframe where it is attached. Bootstrap styling can be applied to this to make it more responsive.



Code Snippet

HTML Code:

The input tag is used to get a youtube link. There is an iframe below the input where the youtube link will be embeded. 

<input id="input_video" type="text" placeholder="Enter the youtube url to add" />
<button id="btn_video" class="btn btn-success">Embed</button>
<iframe id="iframe_video" width="420" height="345" src=""></iframe>

Javascript Code:

Initially the iframe is hidden. When the link is provided, teh iframe is shown and its srcattribute is changed to the provided link.

$(function () {
    $("#iframe_video").hide();
    $("#btn_video").click(function () {
        $("#iframe_video").attr("src", $("#input_video").val());
        $("#iframe_video").show();
    });
});
