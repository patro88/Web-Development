Code Snippet

HTML Code:

The input type is file to accept the files from desktop. On selecting a file , I call a method showMyImage which makes the button select only image type files and also displays a thumbnail image.

<input type="file" accept="image/*" onchange="showMyImage(this)" />
<img id="thumbnil" style="width:20%; margin-top:10px;" src="" alt="image" />

Javascript Code:

This method first checks the file type. If it is not image, we continue. Else I get the thumbnail image and put it in the div below the input.

function showMyImage(fileInput) {
    var files = fileInput.files;
    for (var i = 0; i < files.length; i++) {
        var file = files[i];
        var imageType = /image.*/;
        if (!file.type.match(imageType)) {
            continue;
        }
        var img = document.getElementById("thumbnil");
        img.file = file;
        var reader = new FileReader();
        reader.onload = (function (aImg) {
            return function (e) {
                aImg.src = e.target.result;
            };
        })(img);
        reader.readAsDataURL(file);
    }
}
