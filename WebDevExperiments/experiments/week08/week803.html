HTML Code:

The input is a normal text input. On click of the button, I call the DoSearch method which calls the google api and displays the results below.

<input type="text" id="secrchBox" name="searchterms">
<input type="button" onclick="DoSearch()" value="Search">

Javascript Code:

First we get get a searchcontrol object from google api. Then the searchControl is drawn on the div searchcontrol. After this, teh searchControl execute method is called which hits the google search api and the results are pasted in the given div.

google.load("search", "1", { "nocss": true });
function DoSearch() {
    var ss = document.getElementById("secrchBox").value;
    // Create a search control
    var searchControl = new google.search.SearchControl();
    searchControl.addSearcher(new google.search.WebSearch());
    searchControl.draw(document.getElementById("searchcontrol"));
    // execute an inital search
    searchControl.execute(ss);
}
