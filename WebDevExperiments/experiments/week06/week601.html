Node JS on Openshift : Config

Description:

This week I will be working on Node JS , Express JS and host my application on openshift. In this experiment I learnt to configure openshift application using NodeJs and Express JS. I have returned a simple text on click of a link shared by the server.js on the openshift.
I followed following steps for the configuration:

  1. First, I created an account on Open Shift.
  2. Then I installed the other pre-requisites,
    • Ruby
    • Git Client for Windows
    • Node JS(npm)
  3. Then I installed rhc client using command : gem install rhc on the console
  4. After that, I created a Node JS application on the openshift Application Types Page
  5. After the appplication, was created , I got the ssh address and cloned the app on my local machine using git clone "address"
  6. With the app downloaded, I changed the server.js file with following lines:
    • var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
    • var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
    • app.listen(port, ip);
    Description:
    • The default address and port are the opensift ones, but if the id is not accessible, like in the case of local testing, the local ip and port numbers are used.
