A Simple Node JS application

Description:

In this experiment I have written a simple Node JS application using Express JS. I returned a string from the server when a particular url is hit on the client. The server is configured to receive get requests. The request and response are handled on the server using req and res objects.

Code Snippet

Server.js Code :

In server.js , I defined the express application by following lines of code.

var express = require('express');
var app = express();

A get url is configured to receive the /hello requests. The response object is sent back with a string attached to it, which is displayed on the client.

app.get('/hello', function (req, res) {
res.send("Hello From ExpressJS.");
});

As defined in the earlier experiment , the app is configured to listen to following port and ip. The port and ip are default Openshift node js port and ips, which when not available will default to localhost.

var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
app.listen(port, ip);
