Angular JS - Movie List

Description:

In this experiment, I have have tried implementing a simple retrieval of data from json object in scope of angular js. The data is hard coded by me. I have used the ng-repeat property to display repeated data of the same type. Also, I have used the bootstrap styles.

Code Snippet


<tr ng-repeat="movie in movies">
 <td>{{movie.title}}</td>
 <td>{{movie.year}}</td>
 <td>{{movie.plot}}</td>
 <td>{{movie.genre}}</td>

Note the use of the angular parameter ng-repeat . This is used to iterate over the list of movies in the scope of the model
