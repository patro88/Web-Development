Project : Search Functionality

The search functionality is a very important requirement in any question and answer API. This allows users to search their queries very efficiently. In this project I have implemented very naive search functionality based on keyword search in database. I have used angular $or and javascript Regex functionalities together to achieve this.


Snapshot:

When the user enters a keyword, the search quer returns all questions with title or all questions with body having the same keyword or pattern.



Search Services

Search Services use mongoDB and NodeJs functionalities having following services.

  • SearchService for all questions satisfying the regular expression.
