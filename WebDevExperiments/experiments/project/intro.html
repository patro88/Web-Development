Introduction


Scribal Utility is my project under the course Web Developemnt (CS-5610) that I took during my graduation. Under this utility user get many useful widgets which can be arranged on the website and each widget has different funtionality. Along with the widgets, there is Question and Answers sections where the user can ask questions and get reviews.

List of available functionalities and there uses :

FUNCTIONALITIES USE
Sticky Note Multiple sticky notes can be added. Any type of classroom notes and imporatant point to remember can go into this.
Bookmarks A user can add multiple bookmarks to important sites related to a topic.
Youtube Embed A user can add multiple videos from professor's site or other youtube videos related to a topic.
Image Embed This widget helps to add any image onto the site and can be used for future reference.
Ask Questions This functionality helps the logged in user to ask any questions related to a topic and tag it.
Search Questions This functionality helps any user to search desired questions in the database and get their answers.
Tags Tagging feature helps to group related questions together and their easy retrieval.
Like Question This functionality helps the user to like a question and refer to it later.
View Other users This functionality helps the user to view other users and see thier answered questions.

Different Pages of Project :

PAGE DESCRIPTION
Home Page This is the starting page of the application breifing all the functionalities.
Login and Register Modal This is a modal box used for logging in and registering to use all the features of the wesite.
Profile Page This is the page where a logged in user can create pages and see all his question answer details.
Create Widget Page This the page where user can create all the widgets viz notes, bookmarks, videos and images related to a particular page
Ask Questions Page This is the page where the logged in user can ask his questions
Questions Page This page displays the questions as reuested by the user
Question Details Page This page displays the question in detail and all its answers. User can like or dislike a question here.
Tags Page This page displays all the tags created in the website for easy access to questions.

Technologies used:

TECHNOLOGY PURPOSE USED IN
HTML All the pages and widgets are designed using HTML Client Side
CSS / Bootstarp This is use to enhance the look and feel of project and bootstarp also makes it responsive Client Side
Angular.JS This plays major role in the front end. All user interactions are handled by this. Client Side
Node.js This technology is the backbone of server side functionality. Server Side
Express (Node.js Module) This helps in establishing web services on which the project work. Server Side
MongoDB This is the backend used for data storage purpose Data Storage
Mongoose (Node module) Helps in integrating MongoDB at server side Server Side
