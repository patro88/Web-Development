Project : Widgets Page

The widgets page allows user to create different widgets based on the classroom requirements. following four types of widgets can be created.

  • Note Widget.
  • Bookmark Widget.
  • Embed Media Widget.
  • Embed Image Widget.

Widgets Snapshot

Buttons allow to create different widgets. These are arranged in columns so that widgets appear below each column without disruting the flow.



Note Widget Snapshot :

User can create notes and edit them by clicking on edit button. Both title and body are editable. Gives flexibility for the user to save things.



Bookmarks Snapshot :

User can save bookmarks in his own titles. Clicking on the inks will open the page in a new window.



Media Widget Snapshot :

User can also save the youtube videos of lectures and can watch them when needed. The video is embeded in iframes which allows fullscreen functionality. Also, the ful URL is needed as logic is writtten to slice the url and get the last code. Note that only youtube videos can be saved here.



Image Widget Snapshot :

Images are also stored. This functionality is in progress and will be completed soon.



Widget Services

The services for creating and saving widgets are created using NodeJS and are hosted on openshift. The following sevices are playing a role on this page:

  • Create a widget (all 4 types).
  • Edit a widget (possible only for notes,, bookmarks and video)
  • Delete a widget (possible for all widgets)
