Project : Login and Register

The Login and Register is implemented using a modal in the home page itself. It serves following purposes.

  • Allow user to login to his account to access his personal pages.
  • Allow registration of user into the application to access the full version of the API.
  • This module also expects minimal details from user to register for a hassle free registration.
  • As most of the features like asking a question, creating own pages, and answering a question require a login, this module is of utmost significance.

Login Services

Login services are designed using Node.js and Express Module. MongoDB is used for storing all the user data. Following services are playing role on login page:

  • Login Service - Checks the user credentials, matches them in data base and respond back accordingly



Register User Services

The services for registering a new user are created using NodeJS and are hosted on openshift. The following sevices are playing a role on this page:

  • Check if user name exists or not.
  • Register a new user, store all the details in DB and respond back accordingly
